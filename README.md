# BestHammer

Dont let people put discord invites in your chat without your premission, this bot will warn, kick them, and if necessary, give a ban.

              COMMANDS
              
-------------------------------------------

.help

.warn list

.kicked list

.warn clear

.kicked clear

.addwarn <USER_NAME>

.addkicked <USER_NAME>

.unwarn <USER_NAME>

.unkicked <USER_NAME>
